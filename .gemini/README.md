# .gemini 디렉토리 구조

이 디렉토리는 Gemini CLI 에이전트의 설정을 정의합니다.

## 파일 설명

| 경로 | 설명 |
|------|------|
| `settings.json` | Gemini CLI 도구 설정, 샌드박스 모드, 디렉토리 접근 범위 정의 |

## 관련 파일

| 경로 | 설명 |
|------|------|
| `/GEMINI.md` | Gemini 에이전트의 전역 행동 지침 및 페르소나 정의 |
